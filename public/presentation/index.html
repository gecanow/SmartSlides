Help

The SmartSlidesBegin app is a helpful way to automatically walk you through uploading your .key, .ppt, or .pptx presentation into our system.
It can be found in the SmartSlides project folder, which you should already have. (It's how you're seeing this webpage! If you don't have the project folder, you can download it here.)
 For your convienience, we've also hid all other files in the SmartSlides project folder. To reveal them, press Command + Shift + Period.


To run SmartSlidesBegin, double-click on SmartSlidesBegin.app in Finder.
If you run into trouble using the app, you can also convert your presentation manually by:
 - Opening it in Keynote and exporting it to HTML
 - Dragging the assets folder into this project's public/presentation/ directory
 - Reloading localhost:8000

If see your presentation thumbnails displayed in Step 2, it has been successfully uploaded.
